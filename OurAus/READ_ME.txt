In order to run the Angular Project, do the following:

STEP 1: Download from link below - make sure its finished installing before step 2

https://nodejs.org/en/download/

STEP 2: open  up “Powershell” and type in:

npm install -g @angular/cli


STEP 4: Now open up powershell, and type the folder location for the project

example

cd Downloads/OurAus

STEP 5: Now run the program with following command in powershell

ng serve -o

Will take some time

STEP 5: Open another powershell and go to the same location and go to the backend folder, example below

cd Downloads/AusGov/backend

STEP 6: Type in the following command in the new powershell window to run the server

node server.js
